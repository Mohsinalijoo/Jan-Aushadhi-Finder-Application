# Jan-Aushadhi Finder

Android internship project built with Kotlin and Jetpack Compose.

## Open In Android Studio

1. Open Android Studio.
2. Choose **Open**.
3. Select this folder:

```text
JanAushadhiFinder
```

4. Wait for **Gradle Sync** to finish.
5. Run the app on an emulator or Android phone.

## Main Screens

- Login
- Dashboard
- Brand-to-generic medicine search
- Nearby Jan-Aushadhi Kendra locator
- Stock request simulation
- Refill reminders

The store locator is simulated so the app runs without a Google Maps API key. You can later replace it with Google Maps Compose if your teacher requires a real map.
